# 📝 Cheatsheet PowerShell 

### 📦 Comandos Básicos

- **`Get-Command`**: Obtiene todos los cmdlets, funciones, alias y aplicaciones disponibles en la sesión actual.
- **`Get-Help`**: Proporciona información detallada sobre cmdlets, funciones y scripts.
- **`Get-Alias`**: Muestra una lista de alias definidos en la sesión actual.

---

### 🖼️ Comandos de Imágenes

- **`Export-Csv`**: Exporta objetos a un archivo CSV.
- **`ConvertTo-Html`**: Convierte objetos en una representación HTML.

---

### 🔍 Comandos de Búsqueda

- **`Select-String`**: Busca texto en archivos y cadenas.
- **`Find-Module`**: Busca módulos en el repositorio de PowerShell.

---

### 📅 Comandos de Calendario

- **`Get-Date`**: Obtiene la fecha y hora actuales.
- **`Set-Date`**: Establece la fecha y hora del sistema.

---

### 📧 Comandos de Correo

- **`Send-MailMessage`**: Envía un correo electrónico.

---

### 📊 Comandos de Análisis

- **`Measure-Object`**: Calcula propiedades numéricas de objetos.
- **`Group-Object`**: Agrupa objetos que tienen el mismo valor de propiedad.

---

### 🔒 Comandos de Encriptación

- **`ConvertTo-SecureString`**: Convierte una cadena en una cadena segura.
- **`Protect-CmsMessage`**: Cifra un mensaje.

---

### 📈 Comandos de Monitoreo

- **`Get-Process`**: Obtiene los procesos que se están ejecutando en el equipo local o remoto.
- **`Get-Service`**: Obtiene el estado de los servicios en el equipo local o remoto.

---

### 📤 Comandos de Exportación

- **`Export-Csv`**: Exporta objetos a un archivo CSV.
- **`Export-Clixml`**: Exporta objetos a un archivo XML.

---

### 📥 Comandos de Importación

- **`Import-Csv`**: Importa datos de un archivo CSV.
- **`Import-Clixml`**: Importa datos de un archivo XML.

---

### 🔄 Comandos de Sincronización

- **`Start-Sleep`**: Pausa la ejecución del script durante un período de tiempo especificado.
- **`Wait-Job`**: Espera a que los trabajos en segundo plano se completen.

---

### 📡 Comandos de Conectividad

- **`Test-Connection`**: Envía solicitudes ICMP (ping) a uno o más equipos remotos.
- **`Invoke-WebRequest`**: Envía una solicitud HTTP o HTTPS a una página web o servicio web.

---

### 📦 Comandos de Despliegue

- **`Publish-Module`**: Publica un módulo en un repositorio.
- **`Install-Module`**: Instala un módulo desde un repositorio.

---

### 🔧 Comandos de Mantenimiento

- **`Restart-Service`**: Reinicia un servicio.
- **`Stop-Process`**: Detiene uno o más procesos.

---

### 📂 Comandos de Compresión

- **`Expand-Archive`**: Extrae archivos de un archivo comprimido (ZIP).
